# Number Guessing Pattern Game

A simple Python game that generates an increasing number sequence and challenges the player to guess the last number.

## Description

The program generates a sequence of 5 numbers with a random starting value and a random increment. The first 4 numbers are shown to the player, and the goal is to guess the 5th number.

The game keeps track of attempts and provides feedback until the correct number is guessed.

Example:

```
Sequence: [4, 7, 10, 13]
Guess the last number:
```

## Features

* Randomly generated increasing sequence
* Input validation
* Attempt counter
* Feedback on wrong guesses

## Requirements

* Python 3.x
